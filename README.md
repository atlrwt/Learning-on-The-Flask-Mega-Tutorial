# Learning-on-The-Flask-Mega-Tutorial
A implementation on The Flask Mega-Tutorial

通过本项目，你可以学到：
- Flask的基本使用
- Azure API调用

本项目将用到：
- flask
- Flask-wtf
- flask-sqlalchemy
- flask-migrate
- flask-login
- email-validator
- flask-mail
- flask-bootstrap
- flask-moment
- langdetect
- requests
- python-dotenv
